# Articulate-Storyline-360_Introduction-to-AI

**Technical Stack**

Authoring Tool: Articulate Storyline 360 (HTML5)

Supporting Technology: Animaker, Pixlr, Veo, Nano Banana

Methodology: Iterative Design, Cognitive Theory of Multimedia Learning

Deployment: GitHub Pages

**Project**: Generative AI Integration: Prompt Engineering for Strategic Workforce Productivity

**Project Overview:** This project serves as the first part of a module to improve AI use in the workplace. It addresses the common organizational challenge of "AI ambiguity", where team members have access to powerful tools but lack the structural framework to produce high-value, reliable outputs. By analyzing the gap between "Basic Searching" and "Strategic Prompting," this training provides a roadmap for utilizing Generative AI as a central engine for project management and workflow optimization.

**The Transformation Goal:** To transition a workforce from reactive, low-quality AI interactions to a proactive, standardized Prompt Design methodology that ensures output accuracy, reduces manual revision time, and integrates seamlessly into MS Planner-based project workflows.

**Instructional Architecture**

The first module is built on critical foundational knowledge to develop an understanding of the abilities, limitations and true application potential of AI.  

Process Analysis: Learners experience the milestones of AI development and the subtypes which inform AI capabilities.  

Guided Simulation: A hands-on, risk-free environment where learners construct multi-turn prompts critically, to test ideas against course content.  

Critical Assessment: Active application of ideas to test assumptions of AI in real-time.  Interactive knowledge checks to ensure principle comprehension and application.
